# 1 系统概述

# 1.1 任务依据

《技术要求》（合同名称：舰船装备智能化测试行建模及诊断方案设计技术研究）；

《预研合同》（合同名称：舰船装备智能化测试行建模及诊断方案设计技术研究）；

# 1.2 系统功能

测试性建模与交互式故障诊断一体化集成开发环境软件可实现软件的系统集成、基本器件库、诊断、单点故障定位功能。系统集成功能可以设置各类设备建模与集成建模，如机械设备、电气设备、液压器件等。基本器件库功能可用对电气器件进行查找、编辑、删除、复制，以及对基本器件库的查看、非标准器件模型建模与校验。诊断功能用于进行交互式诊断并生成交互式诊断方案。单点故障定位功能用于对液压泵站启动功能故障进行定位。
