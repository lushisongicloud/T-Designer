# 1 组成和布置

某水声装置释放回收系统由收放控制机柜、电源滤波机柜、收放绞车、固定托架（甲方提供）、导引口（甲方提供）、便携式操作盒等几部分组成。

其中收放控制机柜、电源滤波机柜安装在舱室内，收放绞车、便携式操作盒、固定托架等安装在甲板非飞溅区。

某水声装置释放回收系统的整体安装布置如图 Error: Reference source not found.1 所示。

图 Error: Reference source not found.1 某水声装置释放回收系统安装布置图

各部分的功能如下：

# a) 收放控制机柜

其包括变频器、PLC控制器等，其主要功能是通过便携式操作盒控制，输出驱动绞车电机运转的电源；

# b) 电源滤波机柜

其内部主要为滤波器相关组件，主要功能是抑制及补偿变频器的电磁干扰，使收放装置满足相关电磁兼容要求；

# c) 收放绞车

收放绞车由底座、卷筒、卷筒变频电机、排缆变频电机、转向箱、行星减速器、排缆机构、刹车装置等部分组成，能够实现拖缆、线阵及尾绳的收放及存储和拖曳功能；

# d) 导引口

其位于甲板最外侧，线阵及拖缆通过导引口后入水，该导引口用于缆阵收放时的导向，定位，同时在该导引口上还安装了冲淡水装置和抹水装置，用于缆阵回收时的冲洗和擦干功能；

# e) 固定托架

其位于导引口和收放绞车中间，用于防止拖缆、线阵收放过程中的下垂；

# f) 便携式操作盒

其用来操控绞车运转，同时显示适配柜发送过来的环境参数信息及收放装置工作状态信息。

# 2.5刹车装置

绞车的拖曳刹车装置由手动带式制动器、手动安全插销、电机电磁抱闸三部分组成。

手动带式制动器、手动安全插销安装在卷筒的左侧，电机电磁抱闸集成于电机尾部，采用多重刹车，在结构上实现冗余设计，更加可靠。

# 2.5.1 手动带式刹车

手动带式刹车采用手轮驱动，刹车时操作人员转动手轮，驱动螺杆，该螺杆和其他外部机构组成一个加力杠杆机构，该机构拉动刹车带的两端，使其抱紧卷筒上的刹车鼓，达到刹车目的。

![](images/a6f23e6f7a8db967bd82fc81dbbe9bd13cc310019e09fd0d4f771401660fa4b9.jpg)  
图 Error: Reference source not found.18 手动带式制动器

带式刹车最大制动力矩  $120\mathrm{kNm}$ ，按最大制动力  $94\mathrm{kN}$  计算，卷筒在拖曳时的最大扭矩为  $47\mathrm{kNm}$ ，安全系数为  $\mathrm{n} = 120 / 47 = 2.5$ 。

# 2.5.2 安全插销

手动安全插销为安全保险措施，当刹车完成后，人工插入插销，一旦刹车打滑，插销可以挡住卷筒上的筋板，防止缆绳被连续拉出。在安全插销位设置了一个行程开关，用于安全信号提示保护。其结构如图 Error: Reference source not found.19 所示。

# 2.5.3 电机电磁抱闸

电机电磁抱闸原理如图 Error: Reference source not found.21 所示。当制动线圈接通电源时，其得电，制动盘与制动端盖分开，电机正常转动；当制动线圈接断电时，压力盘在弹簧力作用下移动，制动盘紧贴制动端盖，电机抱闸制动。

为避免超载拖曳力导致电机电磁抱闸机构损坏风险，增加其使用寿命，当电机抱闸制动停车后，推荐使用手动带式刹车进行刹车。

# 3.3.2.3 制动电阻功率计算

在起重变频调速系统运行中，当停车或下降时，重物产生的位势负载使电机处于发电状态，能量向电源侧回馈，由于大多数变频器没有电能回馈装置，此时必须通过制动单元将这部分能量由制动电阻以热能的形式释放掉，所以制动单元和制动电阻在起重变频调速系统中起着非常重要的作用。

某水声装置释放回收系统属于典型的起重变频调速系统，因此其也需要通过制动电阻，来控制、消耗缆阵、拖缆释放所产生的能量。

当拖缆、缆阵下降时，某水声装置释放回收系统变频电机作为发电机产生电能，而变频电机的驱动是来自于拖缆、缆阵的势能，根据能量守恒定律，系统内部产生的电能应等于重物势能的释放，除去系统内部损耗，其他能量都转为电阻的热能耗。所以需计算重物势能产生的功率。

制动电阻的最大发热功率为  $26.04\mathrm{kW}$ ，则每个控制卷筒电机的变频器中的制动电阻的发热功率为  $13.02\mathrm{kW}$ 。

考虑到某水声装置释放回收系统，在释放、回收过程中，其制动电阻的发热功率逐渐增加，并考虑到制动电阻允许一定的过载，因此，最终选取江苏博邦电气的BAR-3000W100ΩJM制动电阻，其功率为  $3\mathrm{kW}$ ，阻值为  $100\Omega$ ，每个变频器并联四个制动电阻，制动电阻总功率可达到  $12\mathrm{kW}$ ，其并联总阻值为  $25\Omega$ （不低于手册要求的  $20\Omega$ ）。

# 4.1.1 危险等级划分原则

研究装备安全性是从研究危险开始的。通过科学的分析，识别潜在危险，并做出

定性和定量的评价，提出在设计、制造和使用装备中消除潜在危险或控制这些危险使之降低到可接受程度的措施，达到安全的目的。

危险严重性是描述某种危险可能引起的事故的严重程度，它是危险风险评估的一个重要参数。危险严重性通常有定性和定量两种评估值。危险严重等级是一种定性的危险程度量度，依据危险事件将会造成事故的人身伤亡和设备损伤程度分为四级，如表Error:Reference source not found-4所示。

# 4.1.2危险源分析

影响某水声装置释放回收系统安全性的可能因素主要来自以下几个方面：

(1) 安全插销未松开时电机运转；  
(2) 手动带式刹车未松开时电机运转;

# 4.2 可靠性设计

# 4.2.1 设备可靠性模型

为简化可靠性数学模型，假设某水声装置释放回收系统中，各分系统只具有正常和失效两种状态、故障相互独立、寿命服从指数分布。

则设备可靠性数学模型为：

$$
R _ {s} = \prod_ {j = 1} ^ {m} R _ {j}
$$

$$
R _ {j} = e ^ {- \lambda_ {j} t}
$$

$$
M T B F = \frac {1}{\lambda_ {s}} = \frac {1}{\sum_ {j = 1} ^ {m} \lambda_ {j}}
$$

式中，MTBF为平均故障间隔时间， $m$ 为设备所含的分系统数， $\lambda_{j}$ 、 $R_{j}$ 分别为第 $j$ 个分系统的失效率和可靠度指标， $\lambda_{s}$ 、 $R_{s}$ 分别为产品的失效率和可靠度。

# 4.4环境适应性

# 4.4.1 寿命期环境组成分析

某水声装置释放回收系统寿命期环境由后勤阶段环境和使用（作战）阶段环境组成，在后勤阶段经历的事件性质与使用（作战）阶段经历的事件性质有较大的区别，

经受的环境种类和强度以及环境对产品的作用方式和影响机理各不相同。

# (1) 后勤阶段环境

设备在后勤阶段运输和贮存前后过程中均要反复进行装卸搬运作业，搬运作业特点是发生的次数多，但每次所用的时间与运输和贮存的时间相比要少很多。

某水声装置释放回收系统搬运过程中主要经受冲击和振动这一类机械环境，搬运主要由机械设备完成。

某水声装置释放回收系统采用公路运输方式，并且按《军用装备包装、装卸、贮存和运输通用大纲》封闭在防雨防尘介质中，振动和冲击的影响最大，高低温、高湿度的影响次之。

某水声装置释放回收系统采用仓库贮存方式，其特点是不受太阳辐射、风和雨的影响，有害的环境因素主要有温度、湿度和霉菌。

# (2) 使用阶段环境

某水声装置释放回收系统的大气环境为海洋环境。收放控制机柜、电源滤波机柜为一般舱室设备；收放绞车、便携式操作盒、固定托架、导引口为甲板固定设备。

舰船停泊在港口或锚地时，对某水声装置释放回收系统产生影响的主要环境因素有温度、湿度、盐雾、霉菌。

舰船在航行时，对拖曳收放装置产生影响的主要环境因素有温度、湿度、盐雾、振动、冲击、霉菌。

# 4.4.2 环境适应性设计准则

环境适应性设计目前有两种不同的设计准则：第一种设计准则也是最常用的设计准则，即根据特定平台环境要求来设计产品，其优点是最大限度地利用资源；第二种设计准则就是所谓的“极限”准则，该准则不是基于特定平台环境要求，而是基于最大设计和制造能力，其优点是设备的环境适应性很强，缺点是提高了设备的费用。

某水声装置释放回收系统环境适应性设计以第一种设计准则为主，即按照装置的

环境要求进行环境适应性设计。

为了保证某水声装置释放回收系统环境适应性满足拖曳收放装置舰的要求，从以下几个方面进行环境适应性设计。

(1) 适当的设计裕量（耐环境裕量）；  
(2) 选用耐环境能力强的零部件、元器件和材料;  
(3) 采取环境防护措施，如使用保护涂（镀）层，进行密封设计等。

c) 对滚动轴承等采用了密封结构设计，通过密封件隔绝腐蚀性水汽对零部件的影响。在各个回转机构上都设计有注油孔，用于日常维护中的加油加脂防护。

# 4.4.3.5 降噪设计

(1) 提升电源滤波机柜、收放控制柜板厚，增强其与内部元器件的连接强度，避免形成共振；  
(2) 柜体设计时, 优化内部风道, 防止引起风流啸叫;  
(3) 内部选用的滤波器、变频器等主要元器件, 噪音均符合 GJB 要求;  
(4) 内部散热风扇，优先选用高频低噪音风扇，降低电源滤波机柜、收放控制柜噪音。

# 4.5测试性设计

(1) 结合系统内部的接近开关、行程开关等各种传感器，并对各变频器进行故障信息的读取，从而对设备进行故障自检；

(2) 电控系统中的元器件及线缆接口等做好标记代号，其与有关图纸及技术文件上的标记相一致，标记代号清晰、明确，便于测试。

# 4.6 保障性设计

(1) 综合考虑图纸资料、备品备件、人员培训、保障设施等因素，制定使用与维修保障方案；  
(2) 编制相应的使用维修说明书，提高使用规程、保养规程、维修指南等内容的可操作性；  
(3) 提供设备全套图纸和模型，为维修、保养等人员提供保障资料；  
(4) 电控系统使用的元器件均采用市场通用的国产商用零部件，保障在后续生产和维修中元器件采购的可持续性；  
(5) 元器件优先选用成熟、市场分量较大的生产制造厂家, 保障能力强。
