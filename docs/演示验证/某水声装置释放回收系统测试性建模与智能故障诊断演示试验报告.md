# 3 专业术语

LRU（line replaceable unit）：现场可更换单元

FDR（fault detection rate）：故障检测率（用规定的方法正确检测到的故障数与故障总数之比)

FIR（fault isolation rate）：故障隔离率（用规定的方法将检测到的故障正确隔离到不大于规定模糊组的故障数与检测到的故障数之比)



# 4 系统简介

项目针对726所某型水声释放回收装置开展测试性建模与智能故障诊断演示试验。系统总体结构如图 26所示，该装置主要包括电源滤波机柜、收放控制机柜、绞车收放机构、排缆机构、导引机构、电机驱动系统、传感器系统等组成部分，主要实现将水声拖体释放入水、拖曳、回收存储等功能，工作环境恶劣、作业流程复杂，装置故障对执行水下探测任务影响极大。

# 5.1 数据准备

建模数据的准备应包括结构组成、故障模式及其属性信息、交联影响关系、测试等内容。数据准备方式包括以下两种：

(1) 根据依据仿真数据采集表格进行梳理准备, 形成建模数据包;  
2）在成品建立的故障模型的基础上，再按照仿真数据采集表格补充关于信号、测试等方面的内容，形成建模数据包。
